check-python-h-first¶

Python advises extension modules include Python.h before other files. This package checks that source files conform to that suggestion.

The intended use is check_python_h_first path/to/header.h path/to/source.c: the script will tell you if the files on the command line include other headers before Python.h. It will also allow certain headers from Pybind or NumPy as the first included header.
